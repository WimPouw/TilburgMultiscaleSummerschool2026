What is SYNAPSIS?

SYNAPSIS is the multimodal body-language science lab at Radboud University. We build open-source tools for video annotation, de-identification, pose estimation and kinematic analysis of social interaction.

  • MaskAnyone — full de-identification platform
  • Masked-Piper — CPU-ready MediaPipe pipeline
  • MaskBench — pose-quality benchmarking
  • EnvisionBox — dataset & tutorial hub
All tools are open-source and GDPR-ready for use with research participants.

Part 1

MaskAnyone

Full-stack de-identification with YOLO + SAM2

MaskAnyone architecture

How it works

  1. Upload — video or folder via UI or API
  2. Detect — YOLOv8-pose finds all people per frame
  3. Segment — SAM2 creates pixel-accurate masks
  4. Apply strategy — blur / pixelate / solid / contours
  5. Export — masked MP4 + pose JSON
GPU required for SAM2. CPU fallback available with MediaPipe segmenter.

Quick start

# 1. Start server (one-time setup)
git clone https://github.com/MaskAnyone/MaskAnyone
cd MaskAnyone
docker compose up -d

# 2. Open UI
open http://localhost:5001

# 3. Or use the API
curl -X POST http://localhost:8080/videos \
  -F "file=@clip.mp4"

First pull downloads ~8 GB model weights.

Part 2

Masked-Piper v2

Modular, CPU-friendly, pip-installable

Masked-Piper v2 — modular pipeline

Three swappable components

frame
  │
  ▼  PoseEstimator    ← mediapipe | vitpose
  │
  ▼  Segmenter        ← mediapipe | sam2-tiny | bbox
  │
  ▼  MaskingStrategy  ← blur | pixelate | solid | silhouette
  │
  ▼
masked_frame + COCO-17 JSON
Runs on CPU — no GPU needed with MediaPipe backend

Install & run

# Install
pip install "masked-piper[mediapipe]"

# Run from CLI
masked-piper process clip.mp4 \
  --strategy blur \
  --estimator mediapipe \
  --segmenter mediapipe \
  --export-coco

# Outputs:
#   clip_masked.mp4
#   clip_pose.json  (COCO-17)

v1 (holistic landmarks → 3 CSVs) is in 01_masked_piper_v1.ipynb

Masking strategies compared

StrategyVisual effectPrivacy level Pose preserved?Use case
blur Gaussian blur over person Medium Yes ✓ General research sharing
pixelate Mosaic / blocky pixels Medium Yes ✓ News / media style
solid Solid colour fill High Yes ✓ Restricted datasets
silhouette Black silhouette + skeleton Highest Yes ✓ Public open data
none No masking — pose only Raw Yes ✓ Internal annotation only
All four strategies preserve COCO-17 keypoints in the JSON export — MaskBench quantifies how well.

Part 3

MaskBench

Does masking degrade body-language signal?

MaskBench evaluation framework

Workflow

Raw video ──► Pose estimator ──► Keypoints_GT ┐
                                               ├─► Compare
Masked video ► Pose estimator ──► Keypoints_M  ┘

Metrics:
  RMSE      — pixel error (lower = better)
  PCK       — % correct keypoints
  Velocity  — motion magnitude similarity
  Acceleration — signal dynamics

Raw video keypoints = ground truth.
Masked video keypoints = prediction.
